# 🎭 AI嘚吧嘚

一个有趣、好玩的AI写作助手，不是来挑错的，而是来陪你聊天、给灵感的创作伙伴。

![React](https://img.shields.io/badge/React-18-blue?style=for-the-badge&logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=for-the-badge&logo=typescript)

## ✨ 特点

- **不爹味** - 没有说教的强制性建议，而是朋友式的友好讨论
- **有个性** - 8个预设AI角色，每个都有独特的人设和说话风格
- **可定制** - 支持自定义AI角色，打造专属的评论团队
- **双模式** - 既可以全文评论，也可以选中部分文字获取建议
- **实时互动** - 基于Tiptap的流畅编辑体验
- **基础纠错** - 内置纠错小助手，温和指出错别字、语病、敏感词等问题

## 🎭 预设AI角色

1. **纠错小助手** 🔍 - 专注基础质量把关（错别字、语病、标点、敏感词等）
2. **产品老王** 🧔 - 关注用户体验，喜欢说"接地气"
3. **文案小美** 💁‍♀️ - 追求有趣有料，会用网络热梗
4. **杠精小李** 🤓 - 温和地挑刺，"emmm虽然但是..."
5. **爱讲涩话的Tim** 😏 - 幽默风趣，适度开车
6. **鸡汤大师** ✨ - 专门夸你和鼓励
7. **严谨学究** 📚 - 关注数据准确性，但不强制
8. **读者视角** 👥 - 代表普通读者的声音

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置API Key

项目使用 [OpenRouter](https://openrouter.ai/) 作为AI接口，支持多种模型（Claude、GPT等）。

1. 访问 [OpenRouter](https://openrouter.ai/keys) 获取API Key
2. 启动项目后，点击右上角的设置按钮 ⚙️
3. 填入你的API Key
4. 选择想要使用的模型（推荐 Claude 3.5 Sonnet）

### 3. 启动项目

```bash
npm run dev
```

访问 `http://localhost:5173` 即可使用。

## 📖 使用指南

### 生成标题

1. 在编辑器中写一些内容（至少50字）
2. 点击标题栏右侧的「✨ 生成」按钮
3. AI会生成一个吸引人的标题，并告诉你为什么推荐这个标题
4. 可以选择「😍 就用这个！」或「🎲 换一个」

### 全文评论

1. 写完文章后，点击右侧面板的 🔄 按钮
2. AI角色们会依次阅读你的文章，给出各自的评论
3. 评论会逐个浮现，每个角色都有独特的视角
4. **纠错小助手会优先检查文章的基础质量问题**

### 选中文字建议

1. 用鼠标选中编辑器中的一段文字（至少10个字）
2. 会浮现一个「✨ AI嘚吧嘚」按钮
3. 点击按钮，右侧评论区会显示AI们针对这段话的建议
4. 建议会统一显示在右侧评论区，和全文评论使用相同的展示方式
5. 前3个AI角色会依次给出建议，包括改写建议

### 自定义角色

1. 点击右侧面板的 ⚙️ 按钮
2. 点击「➕ 添加自定义角色」
3. 填写角色信息：
   - **名称**：例如"爱讲涩话的Tim"
   - **头像**：选择一个Emoji
   - **人设**：描述角色的性格和说话风格
   - **评论风格**：用逗号分隔的风格标签

## 💡 使用技巧

1. **定制你的角色团队** - 根据你的写作类型，选择最适合的AI角色
2. **选中关键段落** - 对不确定的段落点击"AI嘚吧嘚"按钮获取建议
3. **多次生成标题** - 如果不满意标题，可以多点几次"换一个"
4. **优先看纠错** - 纠错小助手会优先指出文章的硬伤问题
5. **保存好的建议** - 评论区的建议可以复制保存，便于后续参考

## 🎯 开发理念

这个项目的核心理念是**陪伴式写作**，而不是纠错式写作：

| ❌ 传统纠错工具 | ✅ AI嘚吧嘚 |
|---------------|-----------|
| "你这里错了" | "这里感觉可以这样试试~" |
| 红色标注错误 | 温暖的评论卡片 |
| 强制修改 | 提供参考，由你决定 |
| 冰冷的工具 | 有趣的伙伴 |

### 文章质量红线检查

纠错小助手会温和地指出以下问题：

1. **错别字和同音错字** - 例如"在做"误写成"在坐"
2. **语病** - 主谓不一致、成分残缺、搭配不当等
3. **标点符号误用** - 顿号、逗号、引号等使用不当
4. **敏感词汇** - 可能引起争议的表达
5. **基本事实错误** - 常识性错误
6. **表达不清** - 逻辑混乱、指代不明

但纠错小助手**不会**挑剔：
- 个人文风和创意表达
- 修辞手法的使用
- 情感化的语言
- 网络用语和梗

## 🔐 隐私说明

- 所有数据都存储在浏览器本地（localStorage）
- API Key仅用于调用OpenRouter服务
- 文章内容通过OpenRouter发送到选择的AI模型
- 不会收集或上传任何用户数据到第三方
