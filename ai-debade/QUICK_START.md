# 🚀 快速开始指南

## 第一次使用

### 1. 配置 API Key

打开项目后，首先需要配置 OpenRouter API Key：

1. 点击右上角的 ⚙️ 设置按钮
2. 访问 https://openrouter.ai/keys 获取你的 API Key
3. 将 API Key 粘贴到设置页面
4. 选择模型（推荐 **Claude 3.5 Sonnet**）
5. 点击「💾 保存设置」

### 2. 开始写作

现在你可以开始使用了！

**写标题：**
- 在顶部输入框输入你的标题
- 或者写完文章后点击「✨ 生成」让AI帮你生成标题

**写正文：**
- 在左侧编辑器中自由创作
- 支持 Markdown 格式
- 至少写50个字才能使用AI功能

### 3. 获取AI建议

**方式一：全文评论**
1. 点击右侧的 🔄 按钮
2. 所有AI角色会依次阅读你的文章
3. 评论会逐个浮现在右侧

**方式二：选中文本**
1. 用鼠标选中一段文字（至少10个字）
2. 点击浮现的「✨ AI嘚吧嘚」按钮
3. 前3个AI会在右侧给出针对性建议

## AI角色说明

每个AI角色都有独特的关注点：

| 角色 | 关注点 | 适合场景 |
|------|--------|---------|
| 🔍 纠错小助手 | 错别字、语病、标点、敏感词 | 基础质量把关 |
| 🧔 产品老王 | 用户体验、是否接地气 | 产品相关文章 |
| 💁‍♀️ 文案小美 | 趣味性、网络热梗 | 新媒体文案 |
| 🤓 杠精小李 | 逻辑严谨、数据准确 | 技术/学术文章 |
| 😏 Tim | 幽默风趣、活跃气氛 | 轻松有趣的内容 |
| ✨ 鸡汤大师 | 鼓励夸奖、增强信心 | 需要正能量时 |
| 📚 严谨学究 | 事实准确、引用规范 | 严肃内容 |
| 👥 读者视角 | 可读性、理解门槛 | 面向大众的文章 |

## 自定义角色

不满足于预设角色？创建你自己的AI评论员：

1. 点击右侧 ⚙️ 按钮
2. 点击「➕ 添加自定义角色」
3. 填写信息：
   - 名称：给角色起个有趣的名字
   - 头像：选一个合适的 Emoji
   - 人设：描述角色的性格、说话方式、专业领域
   - 风格：用逗号分隔的评论风格标签

**示例：**
```
名称：技术老铁
头像：🤖
人设：资深程序员，关注代码质量和技术细节。说话直接但友善，喜欢用技术术语，经常说"这个实现有点意思"。
风格：关注技术准确性, 指出性能问题, 代码规范建议, 技术风格友好
```

## 使用技巧

### 💡 提升写作效率

1. **先纠错，再优化**
   - 第一个角色是「纠错小助手」，优先看它的建议
   - 修复基础问题后，再看其他角色的创意建议

2. **选中重点段落**
   - 对拿不准的段落使用选中功能
   - 可以多次选中不同段落，获取针对性建议

3. **多生成几次标题**
   - 标题是文章的门面
   - 点击「🎲 换一个」多试几次
   - AI会从不同角度给出建议

4. **定制角色团队**
   - 根据你的写作类型，删除不需要的角色
   - 添加符合你需求的自定义角色
   - 角色数量影响生成速度，建议保持5-8个

### 🎨 理解AI的建议

AI的建议分为两类：

**评论类（所有角色）**
- 给出观点和想法
- 指出可能的问题
- 提供改进方向

**改写建议（选中文本时）**
- 具体的改写示例
- 显示在💡建议区域
- 可以参考或直接采用

### ⚠️ 注意事项

1. **API调用有成本**
   - 每次生成都会消耗OpenRouter额度
   - Claude 3.5 Sonnet 性价比最高
   - 可以选择便宜的模型用于草稿

2. **生成速度**
   - 全文评论：所有角色依次生成，需要一些时间
   - 选中文本：只有前3个角色生成，速度较快
   - 生成过程中可以继续编辑

3. **隐私安全**
   - 所有数据保存在本地浏览器
   - 清除浏览器数据会丢失配置和内容
   - 建议定期复制保存重要文章

## 常见问题

**Q: 提示"请先配置 API Key"？**
A: 点击右上角⚙️设置，填入OpenRouter API Key

**Q: 生成速度很慢？**
A: 正常现象。每个AI需要独立生成，可以考虑：
- 减少AI角色数量
- 使用更快的模型（如 Claude 3 Haiku）
- 使用选中文本功能（只生成3个角色）

**Q: AI的建议不满意？**
A: 可以尝试：
- 换个模型试试
- 调整AI角色的人设
- 给AI更多上下文（写更完整的文章）

**Q: 想要更专业的建议？**
A: 创建自定义角色，详细描述你需要的专业视角

**Q: 能保存历史评论吗？**
A: 目前不支持。建议：
- 复制有价值的建议到笔记中
- 或者截图保存
